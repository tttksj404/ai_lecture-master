# Day 6: 파운데이션·멀티모달 — 초간단 정리

처음 배우는 사람 기준으로, 핵심만 쉽게 정리했습니다.

---

## 1) 이 수업에서 하는 일 (한 줄 요약)

**글을 넣어서 이미지를 만들고, 그 이미지가 맞는지 AI로 평가하고, 만든 이미지로 또 다른 AI를 학습**합니다.

---

## 2) 꼭 알아야 할 단어 5개

- **파운데이션 모델**: 큰 데이터로 미리 많이 공부해 둔 기본 AI 모델  
- **멀티모달**: 글+이미지처럼 여러 종류 데이터를 함께 다루는 방식  
- **디퓨전 모델**: 잡음(노이즈)에서 시작해 점점 깨끗하게 이미지를 만드는 모델  
- **CLIP**: 이미지와 문장이 얼마나 잘 맞는지 점수로 알려주는 모델  
- **전이학습**: 이미 학습된 모델을 가져와 내 문제에 맞게 다시 학습하는 방법  

---

## 3) 흐름만 이해하면 끝 (실습 4단계)

1. **이미지 생성**  
   - 예: "숲에 앉아 있는 빨간 여우 수채화" 같은 문장을 넣어 이미지 생성

2. **이미지 평가 (CLIP)**  
   - 만든 이미지가 "여우 그림"에 가까운지 점수로 확인

3. **기존 CNN 모델과 비교 (ResNet)**  
   - CLIP과 ResNet 결과를 비교해서 차이 이해

4. **생성 이미지로 학습 (ResNet-18)**  
   - 내가 만든 합성 데이터로 분류 모델을 다시 학습

---

## 4) CLIP vs ResNet, 뭐가 다를까?

- **CLIP**: 내가 직접 쓴 문장과 이미지의 "의미 일치"를 비교할 수 있음  
- **ResNet**: 미리 정해진 클래스(예: 1000개) 안에서만 분류  

쉽게 말해:
- CLIP = "문장 기반으로 유연하게 판단"
- ResNet = "정해진 보기 안에서 선택"

---

## 5) 왜 합성 데이터가 중요할까?

- 실제 데이터를 모으기 어려울 때 대안이 됨
- 데이터가 적을 때 학습을 시작하기 좋음
- 빠르게 실험 아이디어를 검증할 수 있음

---

## 6) 과제에서 추가로 하는 것

- **Stable Diffusion**으로 이미지 생성
- **Positive/Negative 프롬프트**로 결과 품질 조절
- (선택) **Knowledge Distillation**  
  - 큰 모델(Teacher)의 지식을 작은 모델(Student)에게 전달
