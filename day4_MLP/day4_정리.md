## 1. 신경망(Neural Network) 핵심

### 1.1 선형의 한계
- **y = ax + b** (선형 방정식)만으로는 **XOR 문제**를 해결할 수 없다.
- 선형 방정식을 여러 개 중첩해도 결국 **선형**이다.
- XOR 등 비선형 경계가 필요한 문제를 표현할 수 없다.
- → **비선형**이 필요하다.

### 1.2 비선형: 활성화 함수
- **활성화 함수**로 비선형을 도입한다.
- **시그모이드(Sigmoid)**: 출력을 0~1 확률로 변환할 때 사용
- **ReLU**: 가장 널리 쓰이는 활성화 함수
  - x ≤ 0 → 0
  - x > 0 → x
  - 음수를 0으로 만드는 것만으로도 비선형 구현
  - 연산 효율적 + 성능 우수

### 1.3 다층과 역전파
- 비선형을 여러 개 쌓으면 복잡한 문제를 구분할 수 있다.
- 파라미터 업데이트 연산이 느려졌던 문제 → **역전파(Backpropagation)**로 해결
- 편미분을 활용해 gradient를 효율적으로 계산

---

## 2. 학습 시 미분 추적

### 2.1 언제 OFF?
- **중간 평가(Validation)** 또는 **최종 평가(Test)** 시에는 `requires_grad`를 끈다.
- `model.eval()` 사용

### 2.2 왜 꺼도 되는가?
- 평가 시에는 **예측만** 하면 되고, **미분(역전파)이 필요 없다**.
- gradient를 계산하지 않으므로 메모리·연산 절약

## 6. 학습 루프 순서

1. `optimizer.zero_grad()` — gradient 초기화
2. `output = model(input)` — 순전파
3. `loss = criterion(output, target)` — 손실 계산
4. `loss.backward()` — 역전파
5. `optimizer.step()` — 파라미터 업데이트
